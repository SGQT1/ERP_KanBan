# 成品倉庫存狀況看板

## 项目介绍
这是一个 ERP 系统中的成品仓库状态看板，用于实时展示仓库物料的库存、位置、出入库状态等信息，帮助仓库管理人员快速掌握库存情况。

## 技术栈
- 开发语言：C#
- 运行环境：.NET Framework/.NET Core（根据你的项目填，比如 .NET 6）
- 数据库：SQL Server（如果用到了，没有就删掉）
- 前端框架：WinForm/WPF（根据你的项目填）
